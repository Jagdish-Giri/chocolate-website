# chocolate-website

Static, multi-page chocolate shop demo (home, collection, story) built with HTML/CSS/JS. Uses Unsplash imagery for visuals and localStorage for a lightweight cart counter.

## Pages
- index.html — landing with hero, categories, bestsellers slider, newsletter
- collection.html — filterable/sortable catalog grid
- story.html — atelier narrative, commitments, tasting request form

## Run
Open index.html in your browser. All assets are remote; no build step needed.
